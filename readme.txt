Algoritmo PSO (Particle Swarm Optimization) Discreto para o problema do Caixeiro Viajante

Para executar o algoritmo, execute o arquivo 'tsp_pso.py' da seguinte forma:
' python tsp_pso.py 'arquivo_instanica.txt' populacao iteracoes c1 c2'
EX:
python tsp_pso.py instancia.txt 200 100 0.9 1.5

Será imprimido no terminal o melhor fitness de cada iteração, e no final, a melhor solução encontrada.

Use o arquivo 'auto.sh' para executar o algoritmo diversas vezes.
